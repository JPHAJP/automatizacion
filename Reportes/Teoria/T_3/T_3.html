Práctica 3

Construcción de circuitos básicos

Regresar a Teoría

Descripción

En esta actividad se diseño y construyó un tablero eléctrico para montar circuitos, siguiendo la teoría especificada y tomando en cuenta las normas para rieles y canaletas con la finalidad de representar lo más apegado a un tablero real.

Además, en esta práctica se plantean tres circuitos de automatización industrial con la finalidad de conocer los elementos básicos de una caja de circuitos y algunos de los circuitos básicos.

Diseño 3D de caja

Para poder crear el tablero eléctrico, primero se revisó la teoria de construcción de un tablero, esto incluye normas como la DIN 60715 que especifica la forma de los rieles para ajustar los componentes eléctricos y que sea más sencilla su instalación y con un uso seguro.

Archivo de diseño .STEP
Construcción

Para el armado del modelo 3D del tablero eléctrico se realizaron los planos de las figuras y se exporto a dxf para poder mediante un prototipado rápido con mdf y corte láser construir el tablero; se utilizaron elementos como "press and fit" para mejorar el armado. Finalmente se reforzo mediante pegamento blanco y se monto en el aula para poder realizar las siguientes prácticas del curso en este.

Planos DXF
1er Circuito

Conexión de un paro de emergencia y un circuito de retención con relevador.

2do Circuito

Conexión de un circuito dependiente para encender una lampara mediante relevador.

3er Circuito

Conexión de un motor simple con una sola fase y referencia.
